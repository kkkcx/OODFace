# **OODFace**

This repository contains the implementation of the paper "[OODFace: Benchmarking Robustness of Face Recognition under Open-World
Corruptions and Variations](https://arxiv.org/pdf/2412.02479)" by Caixin Kang, Yubo Chen, Shouwei Ruan, Shiji Zhao, Ruochen Zhang, Jiayi Wang, Shan Fu, Xingxing Wei.
